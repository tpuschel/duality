Introduction

Duality is a high-level programming language aiming to unify and simplify various contructs and concepts found in other programming languages. It presents a set of minimal, coherent and orthogonal constructs that lead to highly expressive yet easily understandable programs.

In particular, Duality generalizes the following concepts commonly found in other languages:

  • Static and dynamic typing.

    In Duality, there is no difference between checking a program for correctness before a program has begun evaluating ("at compile time", or static) or after ("at run time", or dynamic).

    Likewise, the representation of an expression in the Duality language does not differ between the checking and evaluating phases. That means that types exist at 'run time' just as much as at 'compile time'.

    As a consequence of this design, types can influence program behavior both at all and uniformly in the static and dynamic dimension.
