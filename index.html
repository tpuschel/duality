Duality is a high-level programming language aiming to unify and simplify various contructs and concepts found in other programming languages.

A list of features in no particular order:

  • Expression-based
  • Value-oriented
  • Garbage collected
  • Statically & dynamically typed
  • Structurally typed
  • Dependently typed
  • Full type inference
  • Made of minimal, coherent & orthogonal constructs
  • Implementation is a set of easy to use C libraries
  • Cross-platform
  • Language server & plugin for VSCode

Interested? Then check out the documentation for a deeper look at the language!
